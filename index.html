Functionality

TF2BD is a standalone application that identifies and calls vote-kicks against known bots and cheaters on your team.

If they are on the other team, it will send a chat message telling the other team to kick their cheater.

This is an ongoing project that will continue to be improved with new features, more powerful detection, and better customization.

This program is completely VAC compliant

It does not modify the game or OS memory in any way. It is only using built-in functionality in the engine, exactly the way it was intended.

More info about how it works can be found in the docs.

Extended functionality

Customizable rules, Steam Web API integration and Discord Rich Precense.

There are several community driven rule-lists and playerlists to help make sure no bots is able to circumvent detection (Found here).

And With Steam Web API enabled you can see extended player-info, such as account age, game & vac bans.
